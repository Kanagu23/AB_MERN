How Bootstrap's Grid Works

Bootstrap divides each row into 12 columns. You tell each element how many columns to take up at different screen sizes using classes like col-sm-*, col-md-*, col-lg-*. The breakpoints are:

Class prefixScreen widthTypical device
col-<576pxExtra small (phones)
col-sm-≥576pxSmall (large phones)
col-md-≥768pxMedium (tablets)
col-lg-≥992pxLarge (laptops)
col-xl-≥1200pxExtra large (desktops)
col-xxl-≥1400pxExtra extra large

3. Responsive Utility Classes

Bootstrap also lets you show/hide elements or change text alignment per breakpoint.
